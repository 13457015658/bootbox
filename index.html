Hello, Bootbox 5!

This is a simple demo page for the in-progress revamp of bootbox.js. The primary focus is to make sure the various helper functions work consistently, to make any required Bootstrap 4 compatibility updates, and to reduce any code duplication that may have crept in over the years.

We've also implemented the ability to set a locale on a per-dialog basis, using the locale option. Locales are also being split out into separate files, hopefully making pull-requests a little simpler. Grunt will be used eventually to combine the individual locale files into one master file, or you can just reference the locales you need.

Dialog

This is a custom dialog. It should function just like version 4.4.x.

Alert

This is an alert. It is also a test of the slightly revamped locale code. This alert uses a custom locale, set with the setLocale function.

Confirm

This is a confirm. If you clicked the "show alert" button above, you'll see that it uses the locale set with the setLocale function. Otherwise, it will use the default en locale.

Prompt

This is a prompt. This function probably has the most updates, not the least of which is the ability to prepend some text before the input. In other words, you can use the message option just like any of the other functions. We've also added radio and range as input types, and added code to allow the min, max, and step values to be set on number, range, time, and date inputs.
